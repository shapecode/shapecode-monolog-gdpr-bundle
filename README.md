## Install

Via Composer

``` bash
$ composer require shapecode/monolog-gdpr-bundle
```

Enable the bundle in your kernel:

``` php
<?php
// app/AppKernel.php

public function registerBundles()
{
    $bundles = array(
        // ...
        new Shapecode\Bundle\MonologGDPRBundle\ShapecodeMonologGDPRBundle(),
    );
}
```
